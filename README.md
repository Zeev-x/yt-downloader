# yt-downloader

Pengunduh audio dan video YouTube dengan kualitas tinggi menggunakan Python dan nodejs.

## Fitur

* Mengunduh video YouTube dalam kualitas tinggi
* Mengunduh audio saja (misalnya MP3)
* Mendukung konversi menggunakan ffmpeg
* Sederhana dan mudah digunakan

## Persyaratan

* Python (disarankan versi 3.8+)
* ffmpeg

## Instalasi

1. Pastikan Python sudah terpasang:

   ```bash
   python --version
   ```
2. Pasang ffmpeg dan pastikan sudah tersedia di PATH.
3. Clone repositori ini:

   ```bash
   git clone https://github.com/username/yt-downloader.git
   cd yt-downloader
   ```
4. (Opsional) Buat dan aktifkan virtual environment.

## Penggunaan

Contoh penggunaan dasar:

```bash
npm start
```

## Catatan

* Pastikan koneksi internet stabil saat mengunduh.
* Gunakan alat ini hanya untuk konten yang Anda miliki haknya.
